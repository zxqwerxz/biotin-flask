EPIC Array CG to CpG Match

  • Version 0.01 (08-10-18)
  • Takes an excel file containing ‘All CpGs’ and ‘EPIC Array’ tabs and returns the same file with a new sheet labelled ‘New CpGs’ with all the CpGs correctly matched with the CG IDs.

Directions for Input

  1. Input all required parameters (excel filename, name of EPIC array sheet, CG ID heading in EPIC sheet, chromosome number, chromosome location heading from All CpGs sheet).
  2. ’All CpGs’ tab should say ‘All CpGs’ (with a space) and ‘CpG #’ (with a space) should be the heading of the CpG number column.
  3. Chromosome location column in ‘All CpGs’ sheet should have ‘Chr’ with the chromosome number in front of the location (ie ‘Chr11:65432’).
  4. EPIC array should have it’s own sheet with the correct CG IDs filled in.
